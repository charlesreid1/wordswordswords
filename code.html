What's In It

The Milton Bot Flock is implemented entirely in Python. The code base consists of scripts to set up each Milton Bot Flock account to tweet via Python, and to read the text of each poem and turn it into tweets.

The only external library it needs is the python-twitter library from @bear (Github).

The trickiest part of the Bot Flock was mastering Twitter's OAuth2 authentication method. Fortunately for you, I have covered the required steps in detail in a three-part blog post:
